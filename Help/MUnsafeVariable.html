UnsafeVariable Method

       

method UnsafeVariable

Beschrijving:
Berekent voor elke relevante dimensie (1, 2, ..) het aantal onveilige tabelcellen voor een variabele.

Syntax

UnsafeVariable(VarIndex As Long, Count As Long, UCArray As Long) As Boolean

Arguments

VarIndex
Type: Long
1, 2, ... n_var, index van variabele.
Count
Type: Long
Aantal elementen in UCArray
UCArray
Type: Long*
Array met UCs voor aantal dimensies 1, 2, ... Count
Returns
Type: Boolean
false als een of meer parameters fout zijn, anders true
