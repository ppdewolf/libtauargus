CompletedTable Method

       

method CompletedTable

Beschrijving:
Geeft aan dat de inlezen van de table is klaar. Dit methode moet worden geroepen na een aantal keer SetInTable wordt aangeroepen. Hier wordt ook getest als de ingelezen tabel klopt. (de hierarchies telt op de juiste manier)

Syntax

CompletedTable(Index As Long, ErrorCode As Long) As Boolean

Arguments

Index
Type: Long
Table Nummer (1, 2, ... N)
ErrorCode
Type: Long
Error code die geeft aan wat fout ging. Zie FoutCodeLijst
Returns
Type: Boolean
fout als er iets fout ging anders true.
