SetNumberVar Method

       

method SetNumberVar

Beschrijving:
Specificeert het aantal variabelen; reserveert geheugen voor de administratie van de variabelen.
Roept eerst CleanAll aan, voor het geval er een nieuwe actie plaatsvindt.

Syntax

SetNumberVar(nVar As Long) As Boolean

Arguments

nVar
Type: Long
aantal variabelen
Returns
Type: Boolean
false als n_var onjuist (< 1) of onvoldoende geheugen, anders true
