ComputeCodesToIndices Method

       

method ComputeCodesToIndices

Beschrijving:
Gegeven een array van strings (i.e. code,code,code). Deze methode retorneert de tabel index van deze code. (bijv Totaal generaal staat onder, 0,0,0)

Syntax

ComputeCodesToIndices(TableIndex As Long, sCode As Variant, dimIndex As Long) As Boolean

Arguments

TableIndex
Type: Long
Index van table(1,2, ...N)
sCode
Type: Variant
String array van codes
dimIndex
Type: Long
Array van Table Index
Returns
Type: Boolean
false als er iets fout ging (bijv code niet in tabel), true anders
