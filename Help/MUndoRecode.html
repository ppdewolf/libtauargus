UndoRecode Method

       

method UndoRecode

Beschrijving:
Maakt een hercodering ongedaan. Er kan geen hercodering op een hercodering worden uitgevoerd, een hercodering gaat altijd uit van de basiscodelijst van een variabele. Bij een hiërarchische variabele worden alle codes weer op actief gezet. Een nieuwe hercodering kan ook zonder deze functie eerst aan te roepen uitgevoerd worden.
De resultaten kunnen worden opgevraagd met behulp van:
  • UnsafeVariable
  • UnsafeVariableCodes

Syntax

UndoRecode(VarIndex As Long) As Boolean

Arguments

VarIndex
Type: Long
1, 2, ... n_var, index van variabele
Returns
Type: Boolean
false als de parameter fout is, anders true
