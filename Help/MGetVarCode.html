GetVarCode Method

       

method GetVarCode

Beschrijving:
Geeft type en code van een variabele.

Syntax

GetVarCode(VarIndex As Long, CodeIndex As Long, CodeType As Long, CodeString As String, IsMissing As Long, Level As Long) As Boolean

Arguments

VarIndex
Type: Long
Index variabele (1, 2, ...)
CodeIndex
Type: Long
0 (totaal), 1, 2, ... index van code van variabele
CodeType
Type: Long
1 = elementair, 2 = (sub)totaal
CodeString
Type: String
String van de code (kan leeg zijn bij totaal)
IsMissing
Type: Long
0 = Normale code, 1 = Missing code
Level
Type: Long
Bij hiërarchische: niveau (0 = totaal, 1, 2, ...), bij niet-hiërarchische: 0 = totaal, rest = 1
Returns
Type: Boolean
False als er iets fout ging, anders true
