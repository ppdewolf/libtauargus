SetHierarchicalDigits Method

       

method SetHierarchicalDigits

Beschrijving:
Specificeert de eigenschappen van de codes van een hiërarchische variabele, moet plaatsvinden vóór de aanroep van ExploreFile.
Kan niet samengaan met SetHierarchicalCodelist. De som van de array nDigits moet gelijk zijn aan nPos SetVariable). Het aantal groepjes mag niet meer dan 10 zijn.
De codelijst van een hiërarchische variabele wordt vastgesteld uit het invoerbestand én uit de gegeven hiërarchie.
Bijvoorbeeld voor een code van 5 tekens met verdeling {2,1,2}
  • 12
  • 123
  • 12301
  • 12302
  • 12340
  • 12345
  • 12349
  • 124
  • 12401
  • 12402
  • 12403
  • 12404
  • 12405
Alle 5-cijferig codes stonden in het invoerbestand, de andere zijn daaruit afgeleid volgens de verdeling {2,1,2}. Code 12 is de som van code 123 en 124; code 123 is de som van code 12301 t/m 12349, code 124 is de som van code 12401 t/m 12405.

Syntax

SetHierarchicalDigits(VarIndex As Long, nDigitPairs As Long, nDigits As Long) As Boolean

Arguments

VarIndex
Type: Long
1, 2, ... n_var, index van variabele
nDigitPairs
Type: Long
aantal groepjes met elk een zeker aantal tekens
nDigits
Type: Long
Array van nDigitsPairs elementen met elk het aantal tekens van de hiërarchie.
Returns
Type: Boolean
false als een of meer parameters fout zijn, anders true
