SetHierarchicalCodelist Method

       

method SetHierarchicalCodelist

Beschrijving:
Specificeert de eigenschappen van de codes van een hiërarchische variabele, moet plaatsvinden vóór de aanroep van ExploreFile.
Kan niet samengaan met SetHierarchicalDigits.

Syntax

SetHierarchicalCodelist(long VarIndex As , FileName As String, LevelString As String) As Long

Arguments

long VarIndex
Type:
1, 2, ... n_var, index van variabele
FileName
Type: String
Naam van het bestand met de hiërarchische codelijst; de niveaus worden aangegeven door het aantal LevelStrings aan het begin van een regel. Een generaal-totaal wordt geacht niet aanwezig te zijn, dat wordt door het programma zelf toegevoegd als eerste (lege) code van een codelijst. Bijvoorbeeld:
01
..0101
....0101001
....0101002
....0101003
..0102
....0102345
02
..0201
....020134
....020135
enz.
LevelString
Type: String
Geeft een hiërarchisch niveau aan, bijv. " " of "." of "..", in bovenstaand voorbeeld is het laatste voorbeeld gekozen
Returns
Type: Long
Correct: 1, anders een foutcode (HC_FILENOTFOUND ...). Zie Fout Code Lijst
