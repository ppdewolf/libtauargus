SetVarCodeActive Method

       

method SetVarCodeActive

Beschrijving:
Zet alle onderliggende (afstammelingen) codes van een oudercode van een hiërarchische variabele op (in)actief.

Syntax

SetVarCodeActive(VarIndex As Long, CodeIndex As Long, Active As Boolean) As Boolean

Arguments

VarIndex
Type: Long
1, 2, ... n_var, index van variabele
CodeIndex
Type: Long
0 (totaal), 1, 2, ... index van code van variabele
Active
Type: Boolean
true of false (actief of inactief)
Returns
Type: Boolean
false als parameters fout zijn (bijv. geen hiërarchische variabele, geen oudercode), anders true
