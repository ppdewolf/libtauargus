WriteCSV Method

       

method WriteCSV

Beschrijving:
Schrijft een tabel in het CSV-formaat, direct te lezen in Excel. Elk record vormt een regel in het spreadsheet, de kolomwaarden worden gescheiden door een komma. De labels worden veiligheidshalve omsloten door dubbele aanhalingstekens, ze kunnen immers een komma bevatten. Staan er dubbele aanhalingstekens in dan worden die vervangen door enkele.

Syntax

WriteCSV(TableIndex As Long, FileName As String, DimSequence As Long) As Boolean

Arguments

TableIndex
Type: Long
Index tabel (1, 2, ...)
FileName
Type: String
Naam van het resultaatbestand
DimSequence
Type: Long*
Volgorde waarin dimensies worden behandeld; eerste .. in de laag, voorlaatste in de rij, laatste in de kolom. Dus bij een driedimensionale tabel met DimSequence = [3,2,1] komt Tabeldimensievariabele 3 in de laag-, idem 2 in de rij-, idem 1 in de kolomdimensie
Returns
Type: Boolean
false als er iets fout ging, anders true/i>
