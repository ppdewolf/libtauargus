GetTableRow Method

       

method GetTableRow

Beschrijving:
Geeft de cellen van een tabelrij en berekent hun status volgens de bij SetTable gegeven SafetyRule

Syntax

GetTableRow(TableIndex As Long, DimIndex As Long, Cell As Double, Status As Long, CountType As Long) As Boolean

Arguments

TableIndex
Type: Long
Tabelindex (1, 2, ...)
DimIndex
Type: Long
-1 = kolomvariabele (precies één keer), 0,1,... = index in codelijst van dimensievariabele (0 is altijd het dimensietotaal)
Cell
Type: Double
Inhoud tabelcel, volgens CountType
Status
Type: Long
Zie Cel status
CountType
Type: Long
1 = Response, 2 = Shadow, 3 = Cost
Returns
Type: Boolean
false als er iets fout ging, anders true
