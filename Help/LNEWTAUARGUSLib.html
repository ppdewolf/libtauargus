NEWTAUARGUSLib Library

           

Description

NewTauArgus 1.0 Type Library

Classes

TauArgCtrl
TauArgCtrl Class

Beschrijving:

Die class bevat alleen methodes en events. De gebruiker van deze library kan
  • het aantal variablen zetten
  • de attributen van een variables zetten
  • een microdatafile inlezen en de codelijst daarvan opbouwen
  • een code lijst opbouwen van een hierarchiesch codefile
  • het aantal tabellen zetten
  • de attributen van een tabel zetten
  • de tabellen opbouwen van microdata
  • een tabel direct inlezen
