method SetInCodeList

Deze methode doet hetzelfde wat het methode Explore File doet. De codes wordt ingelezen om de tabel te intellen. Deze methode wordt gebruikt als een tabel wordt direct doorgegeven inplaats van een table die wordt opgebouwd van een micro bestand.

Syntax

SetInCodeList(NumberofVar As Long, VarIndex As Long, sCode As Variant, ErrorCode As Long, ErrorInVarIndex As Long) As Boolean

Geeft het aantal Variabelen

Type: Long
Geeft aan welke variablen wordt gegeven
sCode
Type: Variant
String Array van codes
ErrorCode
Type: Long
een error code. Zie FoutCodeList

Type: Long
Welke Variable het misging
