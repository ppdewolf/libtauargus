ExploreFile Method

       

method ExploreFile

Beschrijving:
Onderzoekt voor elke categoriale variabele welke codes voorkomen, zo wordt de codelijst bepaald. Enige uitzondering: er is een hiërarchische codelijst gespecificeerd; dan moeten de aangetroffen codes per se voorkomen in die codelijst.
Bepaalt voor elke numerieke variabele de maximale en minimale waarde.
In het invoerbestand moeten alle records van gelijke lengte zijn. Als enige uitzondering gelden lege records, die worden zonder waarschuwing overgeslagen.

Syntax

ExploreFile(FileName As String, ErrorCode As Long, LineNumber As Long, VarIndex As Long) As Boolean

Arguments

FileName
Type: String
Naam te onderzoeken bestand
ErrorCode
Type: Long
Fout Code. Zie Fout Code Lijst
LineNumber
Type: Long
Regelnummer waar fout optreedt
VarIndex
Type: Long
Index van de numeriek geachte variabele, die niet numeriek is (0 = nvt)
Returns
Type: Boolean
false als een fout optreedt, de aard ervan staat in ErrorCode, anders true
