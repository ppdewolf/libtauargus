SetVariable Method

       

method SetVariable

Beschrijving:
Specificeert de eigenschappen van een variabele.

Syntax

SetVariable(VarIndex As Long, bPos As Long, nPos As Long, nDec As Long, Missing1 As String, Missing2 As String, IsCategorical As Boolean, VARIANT_BOOL IsNumeric As , VARIANT_BOOL IsWeight As , IsHierarchical As Boolean, IsHolding As Boolean) As Boolean

Arguments

VarIndex
Type: Long
1, 2, ... n_var, index van variabele
bPos
Type: Long
1, 2, ... startpositie in elk record van het invoerbestand
nPos
Type: Long
1, 2, ... aantal posities in elk record van het invoerbestand, max. 100
nDec
Type: Long
0,1,.... aantal decimalen (vooral van belang bij schrijven veilig bestand)
Missing1
Type: String
code voor eerste missing (alleen van belang bij categorale variabele)
Missing2
Type: String
code voor tweede missing (alleen van belang bij categorale variabele)
IsPeeper
Type:Boolean
Geeft aan als de Variable is een Peep Variable
IsCategorical
Type: Boolean
is een categoriale variabele
IsNumeric
Type: Boolean
is een numeriek variabele
IsWeight
Type: Boolean
is een gewicht (moet ook eigenschap IsNumeric hebben)
IsHierarchical
Type: Boolean
is een hiërarchische variabele
IsHolding
Type: Boolean
s een variabele die een holding identificeert; alle opeenvolgende records met een gelijke code worden geacht tot eenzelfde holding te behoren
Returns
Type: Boolean
false als een of meer parameters fout zijn, anders true
Note:Voor categorale variabelen: Als Missing1 leeg is wordt Missing2 gepromoveerd. Als beide leeg zijn wordt false teruggegeven. Als beide gelijk zijn wordt Missing2 geacht leeg te zijn.
Voor niet-categorale variabelen zijn Missings irrelevant.
Slechts één variabele mag de eigenschap IsHolding dan wel IsWeight hebben.
Een variabele met eigenschap IsWeight mag niet (IsCategorical of IsHierarchical) zijn.
Een variabele met eigenschap IsHolding mag niet (IsCategorical of IsNumeric of IsWeight of IsHierarchical) zijn.
