ComputeTables Method

       

method ComputeTables

Beschrijving:
Berekent alle met SetTable en SetTableSafety gespecificeerde tabellen uit het invoerbestand.Alle tabellen worden opgeslagen in het geheugen.
Per tabelcel wordt bijgehouden:
  • Resp (Response variabele)
  • Cost (Cost variabele)
  • Freq (frequentie)
  • Shadow (Shadow variabele)
  • Status
  • nMaxScore (aantal bij te houden hoogste scores)
  • MaxScore (Shadow, ongewogen)
  • MaxScoreWeight (bijbehorend gewicht van ongewogen MaxScores, 0 indien nvt of niet toe te passen)

Syntax

ComputeTables(ErrorCode As Long, TableIndex As Long) As Boolean

ErrorCode
Type: Long
Pointer to ErrorCode. ZieFout CodeLijst

Type: Long
Index is 1,2,....
Returns
Type: Boolean
false als een fout optreedt, de aard ervan staat in ErrorCode, zie lijst, anders true
